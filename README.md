# ALMS-Advanced-Library-Management-system-
This is my first Diploma-Level project as MAJOR PROJECT of my Polytechnic Journey. 
The Key Features:
PDF to Quiz – Auto-extract 25 questions from any PDF using PyPDF2 (only 25 because its made by a polytechnic student for a polytechnic student for their External Exams)
Timer-Based Quiz – 25 questions, 12:30 min timer, auto-submit, detailed results
Automated Fine – 14-day borrowing, ₹5/day overdue fine, reservation queue
Cinematic UI – Navy/Gold theme, Dark/Soft Mode with persistent preference
Role-Based Access + Analytics – Student/Admin separation, 63 test cases, 100% pass rate
